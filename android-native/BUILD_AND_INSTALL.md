# Build and Install Notes

1. Open the Android native project in Android Studio.
2. Make sure the app has notification permission on Android 13+.
3. Build and run on the Samsung phone.
4. Open Grit & Motion > More > Reminder test mode.
5. Tap Check background status.
6. Tap Test in 30 seconds.
7. Lock the phone and confirm the notification appears.

Manual web reminders are still available, but native reminders are the reliability target.
