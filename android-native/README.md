# Grit & Motion Android Native Reminder Starter

v1.5 adds a native reminder engine starter. The goal is to make Get Active reminders reliable even when the web app would be paused.

## Native reminder goals
- Random reminders
- No more than 5 per day
- Awake window
- Minimum gap after movement
- Snooze
- Skip
- Reminder history
- Notification permission
- Test reminder in 30 seconds / 2 minutes

## Current status
This folder contains starter Kotlin files for the native reminder bridge. It is intended to be wired into the Android shell from the earlier native starter.
